# Rekisteröinti- ja kirjautumisjärjestelmä (node.js + MongoDB Atlas)

Yksinkertainen web-sovellus, joka sisältää rekisteröinti- ja kirjautumistoiminnot.

## Ominaisuudet
- Käyttäjän rekisteröinti
- Kirjautuminen
- Salasanojen hajautus (bcrypt)
- Tietojen tallennus MongoDB Atlas
- Yksinkertainen ja selkeä HTML käyttöliittymä
- Node.js ja express palvelin
